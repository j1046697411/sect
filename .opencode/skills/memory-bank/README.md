# Memory Bank Skill 使用指南

## 快速使用

### 搜索记忆
```
用户: 搜索 ECS 相关的记忆
用户: 有没有性能优化的解决方案？
用户: 我的偏好是什么？
```

### 添加记忆
```
用户: 记录这个解决方案
用户: 保存我的偏好：喜欢简洁回复
用户: 这是项目约定，记录下来
```

### 更新记忆
```
用户: 之前的方案过时了，更新一下
用户: 标记那个记忆为废弃
```

### 浏览记忆
```
用户: 列出所有记忆
用户: 显示 solutions 分类下的记忆
用户: 查看某个记忆的详细内容
```

## AI 自动触发

AI 会在以下场景自动使用记忆库：

1. **对话开始** → 自动搜索相关记忆
2. **问题解决** → 提示记录新解决方案
3. **发现偏好** → 提示保存用户偏好
4. **方案变化** → 提示更新过时记忆

## 记忆格式

```markdown
## 标题

> 创建时间: YYYY-MM-DD
> 分类: [conventions|solutions|preferences|lessons]
> 状态: [active|deprecated]
> 标签: #标签1 #标签2

**问题/背景**:
描述...

**解决方案/内容**:
具体内容...

**关联记忆**: #相关记忆ID
```

## 常用操作命令

```bash
# 搜索
grep -r "关键词" docs/memory-bank/

# 列出所有
find docs/memory-bank -name "*.md" -type f

# 查看分类
ls docs/memory-bank/solutions/
```

## 最佳实践

1. **及时记录** - 解决问题后立即记录
2. **准确分类** - 选择正确的分类
3. **简洁标题** - 便于识别和搜索
4. **完整标签** - 提高搜索命中率
5. **建立关联** - 形成知识网络

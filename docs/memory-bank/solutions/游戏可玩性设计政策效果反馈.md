## 游戏可玩性设计：政策效果反馈

> 创建时间: 2026-02-19 11:11
> 更新时间: 2026-02-19 11:11
> 分类: solutions
> 状态: active
> 使用次数: 0

**问题/背景**:
政策调整后效果通过AI行为间接体现，玩家难以建立因果关系，不知道调整带来了什么影响。

**解决方案/内容**:
混合式政策效果反馈设计：
1. 即时预览：调整政策时显示预期效果，包括正面和负面影响
2. 效果追踪：政策生效后持续显示预期vs实际效果对比，显示偏差
3. 对比报告：每月生成政策效果对比报告，包含修炼、资源、弟子状态等维度
4. 触发时机：政策调整时预览、每月追踪、每月报告、偏差>10%异常提醒、满意度<70%预警
5. 信息层级：核心指标顶部显示、详细数据详情页、趋势分析图表、弟子反馈事件日志

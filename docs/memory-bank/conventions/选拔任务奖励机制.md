## 选拔任务奖励机制

> 创建时间: 2026-02-21 22:14
> 更新时间: 2026-02-21 22:14
> 分类: conventions
> 状态: active
> 使用次数: 0

**问题/背景**:
参与选拔任务的长老和内门弟子应该获得什么奖励

**解决方案/内容**:
选拔任务奖励体系：1) 长老奖励 - 贡献点（根据任务难度）、声望提升、培养成就感；2) 内门弟子奖励 - 贡献点、实战经验（微量修为增长）、表现优秀者可被长老关注（未来晋升亲传的机会）；3) 外门弟子无个人奖励，作为资源消耗；4) 任务成功额外奖励 - 发现稀有资源、获得特殊物品

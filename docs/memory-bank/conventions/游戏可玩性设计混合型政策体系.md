## 游戏可玩性设计：混合型政策体系

> 创建时间: 2026-02-19 10:04
> 更新时间: 2026-02-19 10:07
> 分类: conventions
> 状态: active
> 使用次数: 0

**问题/背景**:
核心循环中政策设计过于宏观，玩家缺乏具体可调节的参数，难以感知决策影响。原有设计制定政策到AI执行到观察反馈过于抽象。

**解决方案/内容**:
采用混合型政策设计，分层三层次：第一层战略方向每年调整包含发展重心、资源分配、门派定位、人才培养、外交策略5个参数。第二层战术规则每季度调整包含突破规则、任务优先级、设施使用、奖惩机制、功法传承5个参数。第三层即时干预随时操作包含市场交易、紧急调配、特殊任命、事件响应4种操作。选择标准版每层5到8个参数，平衡深度与可玩性。

## 记忆库命名与标签规范

> 创建时间: 2026-02-23
> 分类: conventions
> 状态: active
> 标签: #记忆库 #规范

**问题/背景**:
记忆库标题命名混乱、标签不一致，导致检索困难。

**解决方案/内容**:

### 命名原则
1. **简洁** - 控制在 10 字以内
2. **可读** - 纯中文，避免中英混杂
3. **可搜索** - 关键词前置
4. **去冗余** - 目录已提供分类，标题不再重复

### 标签层级
```
一级标签（系统域）    二级标签（技术栈）    三级标签（类型）
─────────────────    ─────────────────    ─────────────────
#角色系统             #ecs                 #规范
#经济系统             #kotlin              #方案
#战斗系统             #compose             #陷阱
#功法系统             #测试                #偏好
#设施系统                                  #性能
#可玩性
```

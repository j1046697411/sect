## 选拔任务执行机制-带AI角色

> 创建时间: 2026-02-21 22:13
> 更新时间: 2026-02-21 22:13
> 分类: conventions
> 状态: active
> 使用次数: 0

**问题/背景**:
选拔任务需要哪些角色参与执行

**解决方案/内容**:
选拔任务执行团队：1名长老（领队，负责评估和决策）+ N名内门弟子（执行主力）+ M名外门弟子（辅助资源）。任务完成后，长老根据表现提名候选人，掌门最终挑选晋升者。这样AI角色深度参与选拔过程，长老的性格（公正/偏私）会影响评估结果

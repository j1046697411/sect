## 弟子招募机制

> 创建时间: 2026-02-23
> 分类: conventions
> 状态: active
> 标签: #角色系统 #招募 #长老

**问题/背景**:
弟子招募系统如何设计，包括触发方式、人数计算、资质决定、成本消耗、主持角色等。

**解决方案/内容**:
1. **触发方式**：固定周期招募，周期可配置（如3年/5年）
2. **招募人数**：动态计算 = 宗门等级/规模 + 主持长老能力
3. **资质决定**：根据宗门声望影响弟子资质分布
4. **招募成本**：消耗灵石和食物
5. **主持角色**：长老主持，后台模式（不影响其他任务）

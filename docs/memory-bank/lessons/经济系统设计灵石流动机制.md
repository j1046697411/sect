## 经济系统设计：灵石流动机制

> 创建时间: 2026-02-19 12:23
> 更新时间: 2026-02-19 12:23
> 分类: lessons
> 状态: active

**问题/背景**:
灵石作为外部货币，获取困难会直接影响游戏体验，需要设计完整的获取、转移、消耗流程。

**解决方案/内容**:
灵石流动机制设计：
一、灵石获取（创造）：宗门基础产出每季度50灵石、灵脉产出每季度200-1000灵石、弟子历练随机50-500灵石、探索发现一次性50-500灵石
二、宗门收入（税收）：交易税收1%-5%可调、历练税收10%-30%可调
三、灵石转移：宗门任务奖励、弟子任务（A发布B执行）、工资发放、宗门交易、市场交易、历练税收
四、灵石消耗：设施维护每月、设施建设/升级一次性、突破消耗、外部购买、弟子死亡
五、平衡机制：弟子个人消耗（修炼/丹药/功法/设施/社交）、税收与满意度挂钩、境界消耗递增、贫富差距调节（最低保障/任务优先/遗产继承）
注意：任务不产出灵石，只是灵石转移手段

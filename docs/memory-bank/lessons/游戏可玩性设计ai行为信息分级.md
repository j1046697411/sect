## 游戏可玩性设计：AI行为信息分级

> 创建时间: 2026-02-19 11:48
> 更新时间: 2026-02-19 11:48
> 分类: lessons
> 状态: active
> 使用次数: 0

**问题/背景**:
AI行为信息量大，玩家容易信息过载，不知道该看什么。

**解决方案/内容**:
混合式AI行为信息分级设计：
1. 按重要程度：关键（突破/死亡）始终显示+弹窗、重要默认显示、一般可折叠、日常默认隐藏
2. 按事件类型：修炼/战斗/社交/管理/资源/探索，可按标签筛选
3. 按角色层级：核心（掌门/长老）始终显示、骨干（亲传弟子）默认显示、普通（内门弟子）可折叠
4. 智能推送：关键事件弹窗+音效、重要事件每日汇总、异常情况预警
5. 个性化设置：默认显示级别、关注角色列表、事件类型订阅、推送频率
注意：AI角色只有掌门、长老、亲传弟子、内门弟子四种，其它为可重用资源

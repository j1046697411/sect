# 炼丹房页面布局设计

## 一、设计原则

1. **丹药分类清晰**：将丹药按类型、等级、效果分组展示，便于炼制和管理
2. **核心信息突出**：优先展示丹药的名称、等级、材料需求和效果
3. **炼制流程直观**：清晰展示丹药炼制的条件、材料需求、成功率和时间
4. **操作便捷高效**：提供丹药炼制、加速、停止等快捷操作
5. **炼制队列可视化**：显示正在炼制的丹药和剩余时间
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索、任务大厅、功法堂
- 生产系统：炼丹房（当前选中）、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示炼丹房详细内容
**详细内容**：

#### 3.1 炼丹炉状态
- 显示炼丹炉的名称、等级、成功率加成、当前使用状态和整体状态
- 显示当前炼丹师的名称、境界、炼丹技能和技能加成
- 直观展示炼丹房的核心设施和人员状态

#### 3.2 丹药分类与筛选
- 多种分类方式：全部、恢复、辅助、突破、特殊、毒丹
- 多种等级筛选：凡级、灵级、地级、天级、神级
- 多种效果筛选：回气、疗伤、解毒、聚气、筑基、金丹、元婴、化神
- 便于玩家快速找到符合条件的丹药

#### 3.3 丹药列表
- 按等级展示丹药，包含：丹药名称、类型、等级、成功率、时间、材料需求、效果
- 不同等级丹药使用不同图标和颜色区分
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的丹药
- 提供快捷操作按钮：炼制、查看详情、放弃

#### 3.4 功能标签页
- 丹药列表（当前选中）、炼制队列、炼制记录、材料管理、炉鼎管理
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示炼丹统计和快捷操作
**详细内容**：

#### 4.1 炼丹统计
- 丹药总数
- 已炼制次数
- 成功次数和成功率

#### 4.2 炼丹快捷操作
- 常用操作：炼制丹药、刷新、炼制记录、炼制建议、查看仓库、取出丹药、存入丹药
- 快捷入口，提升操作效率

#### 4.3 炼制队列
- 显示正在炼制的丹药
- 包含队列位置、丹药名称、炼制进度、剩余时间
- 显示等待中和空闲队列

#### 4.4 炼丹材料
- 显示当前炼丹所需的核心材料：灵草、泉水、妖兽内丹
- 显示材料的当前数量和总量

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 四、交互设计

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

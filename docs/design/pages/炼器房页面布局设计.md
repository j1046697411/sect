# 炼器房页面布局设计

## 一、设计原则

1. **装备分类清晰**：将装备按类型、等级、效果分组展示，便于打造和管理
2. **核心信息突出**：优先展示装备的名称、等级、材料需求和效果
3. **打造流程直观**：清晰展示装备打造的条件、材料需求、成功率和时间
4. **操作便捷高效**：提供装备打造、加速、停止等快捷操作
5. **打造队列可视化**：显示正在打造的装备和剩余时间
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索、任务大厅、功法堂
- 生产系统：炼丹房、炼器房（当前选中）
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示炼器房详细内容
**详细内容**：

#### 3.1 熔炉状态
- 显示熔炉的名称、等级、成功率加成、当前使用状态和整体状态
- 显示当前炼器师的名称、境界、炼器技能和技能加成
- 直观展示炼器房的核心设施和人员状态

#### 3.2 装备分类与筛选
- 多种分类方式：全部、武器、防具、饰品、法宝、特殊
- 多种等级筛选：凡级、灵级、地级、天级、神级
- 多种类型筛选：剑、刀、法杖、头盔、铠甲、护腿、戒指、项链、手镯
- 便于玩家快速找到符合条件的装备

#### 3.3 装备列表
- 按等级展示装备，包含：装备名称、类型、等级、成功率、时间、材料需求、效果
- 不同等级装备使用不同图标和颜色区分
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的装备
- 提供快捷操作按钮：打造、查看详情、放弃

#### 3.4 功能标签页
- 装备列表（当前选中）、打造队列、打造记录、材料管理、熔炉管理
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示炼器统计和快捷操作
**详细内容**：

#### 4.1 炼器统计
- 装备总数
- 已打造次数
- 成功次数和成功率

#### 4.2 炼器快捷操作
- 常用操作：打造装备、刷新、打造记录、打造建议、查看仓库、取出装备、存入装备
- 快捷入口，提升操作效率

#### 4.3 打造队列
- 显示正在打造的装备
- 包含队列位置、装备名称、打造进度、剩余时间
- 显示等待中和空闲队列

#### 4.4 炼器材料
- 显示当前炼器所需的核心材料：玄铁、灵草、灵石
- 显示材料的当前数量和总量

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 四、交互设计

### 1. 快捷键系统

| 快捷键 | 功能 | 适用场景 |
|--------|------|---------|
| Shift+9 | 切换到炼器房 | 所有界面 |
| / | 聚焦全局搜索框 | 所有界面 |

| Ctrl+A | 加速打造 | 打造队列 |

- 打造进度实时更新
- 装备打造完成提示

- 顶部导航栏支持展开详情
- 底部状态栏支持折叠/展开

### 2. 动态布局调整

- 长列表支持滚动
- 对话框模态显示
- 自适应不同屏幕尺寸

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

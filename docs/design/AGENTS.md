# 设计文档 - AI 指引

## 目录定位
存放游戏玩法设计及 UI/UX 界面设计文档。

## 核心职责
- 详细定义游戏数值、系统逻辑。
- 提供界面布局和交互设计的详细说明。

## AI 开发指引

- **逻辑一致性**: 业务系统开发必须符合 `sect_cultivation_core_gameplay.md` 等玩法定义。

## 关键内容
- `index.md`: 设计文档索引

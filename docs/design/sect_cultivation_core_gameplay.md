# 宗门修真录 - 核心需求文档

## 1. 项目概述

**一句话描述**：玩家作为宗门总管理者，制定战略政策，观察AI角色自主运转、世代更替的修仙宗门。

### 核心体验
- 宏观管理视角：统筹全局，不做微观操作
- AI自主运转：所有角色根据职位自动处理事务
- 世代更替：老角色逝去，新角色成长，传承延续
- 深度观察：查看每个弟子的详细行为和结果

### 玩家参与度
| 参与方式 | 频率 | 内容 |
|----------|------|------|
| 政策制定 | 每游戏年1次 | 设定发展方向、资源分配原则 |
| 重大决策 | 平均5-10分钟 | 事件触发，选择后果 |
| 市场交易 | 随时 | 通过市场干预门派运转 |
| 深度观察 | 随时 | 查看弟子详情和行为轨迹 |

### 时间比例
- **游戏时间**：1年 = 现实世界6分钟
- **游戏流速**：可调节（暂停/正常/快进）

## 2. 核心玩法循环

```
制定政策 → AI执行 → 观察反馈 → 事件决策 → 调整政策 → [循环]
```

**AI自主运转**：
- 日常资源生产与分配
- 弟子修炼安排与突破尝试
- 设施使用与维护
- 社交关系建立与发展
- 任务接取与执行
- 战斗与试炼参与
- 功法学习与传承

**日常循环**：
- 辰时：弟子起床、晨练
- 巳时：修炼/工作
- 午时：休息、社交
- 未时：修炼/工作
- 酉时：任务/试炼
- 戌时：自由活动
- 亥时：休息

## 3. 角色系统

### 角色类型

| 类型 | 职责 | 晋升条件 |
|------|------|----------|
| 外门弟子 | 基础工作、杂务 | 无 |
| 内门弟子 | 专业修炼、部门工作 | 修为达标 |
| 亲传弟子 | 核心传承、高阶修炼 | 长老推荐 |
| 长老 | 部门管理、弟子指导 | 修为+贡献 |
| 掌门 | 战略决策、对外代表 | 前任指定 |

### 角色属性

**基础属性**：

| 属性 | 影响 |
|------|------|
| 根骨 | 修炼速度、突破成功率 |
| 悟性 | 功法学习速度、领悟能力 |
| 福缘 | 掉落率、奇遇概率 |
| 魅力 | 社交成功率、收入加成 |

**四维属性**：

| 属性 | 影响 |
|------|------|
| 力量 | 战斗伤害、采集效率 |
| 敏捷 | 回避率、修炼间隔 |
| 智力 | 法术伤害、炼丹/炼器成功率 |
| 耐力 | 战斗持久、资源采集量 |

**性格属性**：

| 性格 | 决策倾向 |
|------|----------|
| 野心 | 争取晋升机会、承担风险 |
| 勤勉 | 增加修炼时间、高任务完成率 |
| 忠诚 | 服从管理、少提意见 |
| 贪婪 | 注重收益、可能囤货 |
| 和善 | 积极参与社交、帮助他人 |
| 冷漠 | 独来独往、减少社交 |

### 招募系统
- **自动入门**：每年固定时间开放测试
- **主动招募**：玩家消耗门派资金邀请
- **招募条件**：根骨≥10、年龄≤30岁

## 4. 修炼系统

### 境界体系

| 境界 | 层数 | 寿命加成 | 修炼速度 |
|------|------|----------|----------|
| 炼气期 | 1-9 | +0年 | 基础 |
| 筑基期 | 1-9 | +50年 | ×1.2 |
| 金丹期 | 1-9 | +150年 | ×1.5 |
| 元婴期 | 1-9 | +400年 | ×2.0 |
| 化神期 | 1-9 | +800年 | ×2.5 |
| 大乘期 | 1-9 | +1500年 | ×3.0 |

### 境界解锁顺序
炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 大乘期 → 飞升

### 突破成功率

| 突破 | 成功率 |
|------|--------|
| 炼气→筑基 | 50% |
| 筑基→金丹 | 30% |
| 金丹→元婴 | 20% |
| 元婴→化神 | 10% |
| 化神→大乘 | 5% |
| 大乘→飞升 | 1% |

### 突破结果
- **成功**：进入下一境界1层、获得寿命延长、解锁新功法/技能
- **失败**：停留在当前境界9层、修为倒退10%-30%、受伤需要恢复

### 功法类型
| 类型 | 效果 |
|------|------|
| 主修功法 | 增加对应境界修炼速度 |
| 辅助功法 | 提供特殊能力/属性加成 |
| 战斗功法 | 提升战斗能力 |
| 炼丹/炼器 | 提升生产效率 |

## 7. 俸禄系统

### 俸禄计算

```
基础俸禄 = 职位基数 × (境界系数 + 绩效系数)
```

**职位基数**：

| 职位 | 俸禄（灵石/月） |
|------|-----------------|
| 外门弟子 | 10 |
| 内门弟子 | 30 |
| 亲传弟子 | 60 |
| 长老 | 150 |
| 掌门 | 500 |

**境界系数**：炼气1.0 → 筑基1.2 → 金丹1.5 → 元婴2.0 → 化神3.0 → 大乘5.0

### 士气影响

**发放延迟惩罚**：

| 延迟天数 | 满意度惩罚 | 后果 |
|----------|------------|------|
| 1-7天 | -10% | 轻度不满 |
| 8-14天 | -25% | 工作效率下降 |
| 15-30天 | -50% | 可能离职 |
| 30天+ | -100% | 确定离职 |

### 玩家调控
- **基础倍率**：0.5x-2.0x可调
- **奖金比例**：0-50%
- **特殊津贴**：特定职位额外补贴

## 9. 战斗系统

### 战斗类型

| 类型 | 触发条件 |
|------|----------|
| 妖兽袭击 | 随机事件 |
| 试炼挑战 | 任务/修炼 |
| 外出探索 | 探索触发 |
| 宗门冲突 | 外交事件 |
| 守卫战 | 危机事件 |

### 回合制流程
1. 回合开始
2. 速度判定行动顺序
3. 双方轮流行动
4. 血量归零判定胜负
5. 回合结束

### 行动类型
| 行动 | 效果 |
|------|------|
| 攻击 | 对敌人造成伤害 |
| 防御 | 减少受到的伤害 |
| 技能 | 使用特殊技能 |
| 道具 | 使用丹药/符箓 |
| 逃跑 | 脱离战斗 |

### 伤害计算
```
伤害 = (攻击方攻击 - 防御方防御) × 技能系数 × 属性加成
暴击：伤害 × 2（10%基础概率）
闪避：攻击失效（敏捷差决定）
格挡：伤害 × 0.5
```

### 掉落系统

| 类型 | 概率 |
|------|------|
| 灵石 | 100% |
| 材料 | 60% |
| 装备 | 30% |
| 稀有道具 | 5% |

## 10. 经济与交易系统

### 门派资金

**来源**：
- 税收收入：0%-20%可调
- 交易税收：1%-5%可调

**用途**：设施建设维护、俸禄发放、市场物资买入

### 市场交易

**交易方式**：
- **直接寄售**：挂单出售，等待买家
- **市场购买**：买入现有挂单
- **搜购订单**：发布需求等待供货

**核心原则**：所有交易必须经过市场

### 玩家干预
- **价格调控**：低买高卖
- **资源分配**：通过市场定向扶持
- **税收调节**：平衡资金与士气

## 11. 事件系统

### 事件分类

| 类型 | 示例 |
|------|------|
| 外交事件 | 结盟、宣战、贸易 |
| 危机事件 | 妖兽入侵、资源危机 |
| 发现事件 | 灵脉、洞府、功法 |

### 处理流程
```
触发 → 评估 → 决策 → 执行 → 反馈
```

- **普通事件**：AI自主处理
- **重大事件**：玩家决策

### 触发机制
- **时间触发**：固定周期事件（门派大比）
- **条件触发**：资源/修为/声望达到阈值
- **随机触发**：探索发现、外部入侵

## 12. AI决策机制

### 决策层级

| 层级 | 决策者 | 信息可见性 |
|------|--------|-----------|
| 战略层 | 玩家 | 完全可见 |
| 战术层 | 掌门AI | 决策日志 |
| 执行层 | 长老AI | 部门报告 |
| 操作层 | 弟子AI | 详细行为 |

### 性格与目标优先级

```
高野心 → 地位 > 修为 > 财富
高勤勉 → 修为 > 地位 > 财富
高忠诚 → 门派利益 > 个人发展
高贪婪 → 财富 > 修为 > 地位
```

## 13. 观察系统

### 观察层次
- **宏观层**：宗门整体状态
- **中观层**：部门/长老管理效果
- **微观层**：单个弟子详情

### 观察内容
- 基本信息、属性详情
- 行为记录、决策日志
- 成长轨迹、社交关系

### 预警信号
- 弟子行为异常
- 资源消耗异常
- 关系紧张
- 突破失败率高

### 汇报周期
- 每日简报
- 每周汇总
- 每月深度报告
- 每年全面总结

# 宗门修真录 - 游戏设计总结

> 更新时间: 2026-02-19
> 本文档汇总了游戏各系统的核心设计决策

---

## 一、可玩性设计

### 1.1 政策体系

| 主题 | 核心决策 |
|------|---------|
| **政策体系** | 混合型三层设计（战略/战术/即时） |
| **时间加速** | 5档速度（暂停/1x/2x/5x/10x） |
| **隐形难度曲线** | 修炼时间+成功率+资源+失败后果四维度 |
| **阶段时间** | 初创期30分钟-1小时，发展期2-5小时，繁荣期15-25小时 |
| **补偿机制** | 新系统+新能力+新内容三维度解锁 |
| **里程碑进度显示** | 混合式：阶段面板+进度条+详细列表 |
| **政策效果反馈** | 混合式：即时预览+效果追踪+对比报告 |

### 1.2 AI干预与观察

| 主题 | 核心决策 |
|------|---------|
| **单个AI干预** | 混合式：资源分配+任务指派+特殊培养+奖惩 |
| **AI行为信息分级** | 混合式：重要程度+事件类型+角色层级 |
| **事件频率系统** | 混合式：频率调整+处理方式区分+事件队列 |

### 1.3 长期吸引力

| 主题 | 核心决策 |
|------|---------|
| **多样化目标** | 混合式：多线目标+阶段递进+挑战系统 |
| **开发优先级** | 先实现飞升路线，再逐步扩展其它路线 |

## 二、经济系统

### 2.1 货币体系

| 货币 | 定位 | 核心功能 |
|------|------|---------|
| **灵石** | 核心经济 | 外部交易、资源购买、修炼消耗 |
| **贡献点** | 补充经济 | 内部晋升、功法学习、设施使用 |

### 2.2 灵石流动

| 类型 | 内容 |
|------|------|
| **获取（创造）** | 宗门基础产出、灵脉产出、弟子历练、探索发现 |
| **宗门收入（税收）** | 交易税收1%-5%、历练税收10%-30% |
| **转移** | 任务奖励、弟子任务、工资发放、市场交易 |
| **消耗（消失）** | 设施维护、建设升级、突破消耗、外部购买 |

### 2.3 平衡机制

| 机制 | 说明 |
|------|------|
| **弟子个人消耗** | 修炼、丹药、功法、设施、社交 |
| **税收与满意度挂钩** | 高税率降低弟子满意度 |
| **境界消耗递增** | 境界越高消耗越大 |
| **贫富差距调节** | 最低保障、任务优先、遗产继承 |

### 2.4 玩家参与

| 参与方式 | 说明 |
|---------|------|
| **市场收购/寄售** | 玩家可在市场中买卖物品 |
| **税率调整** | 调整交易税/历练税比例 |
| **历练开关** | 开启/关闭历练场景 |
| **资源分配** | 调整资源分配比例 |

## 三、设施系统

### 3.1 设施分类

| 分类方式 | 内容 |
|---------|------|
| **按功能** | 修炼类、生产类、资源类、管理类、生活类、防御类 |
| **按层级** | 公共、内门、核心、专属 |
| **按阶段解锁** | 初创期、发展期、繁荣期、传承期 |

### 3.2 设施功能

| 设计 | 说明 |
|------|------|
| **多功能设计** | 每个设施有主功能、副功能、特殊功能 |
| **设施联动** | 修炼室+灵脉阵效率+15%、炼丹房+灵药园品质+10% |
| **升级效果** | 效率提升、容量增加、解锁新功能、品质提升 |

### 3.3 设施互动

| 互动类型 | 说明 |
|---------|------|
| **弟子使用** | 普通使用、专注使用、协作使用、指导使用 |
| **资源流动** | 输入资源→输出产品 |
| **时间安排** | 即时使用、短时使用、长时使用、周期使用、预约使用 |

### 3.4 设施平衡

| 机制 | 说明 |
|------|------|
| **功能差异化** | 每个设施有独特功能不可替代 |
| **收益均衡化** | 各设施投资回报率相近 |
| **场景化需求** | 不同阶段/场景需要不同设施 |

## 四、角色系统

### 4.1 属性体系

| 类型 | 属性 |
|------|------|
| **基础属性** | 境界、境界阶段、年龄、性别、气血、灵力 |
| **资质属性** | 根骨、悟性、福缘、体质、神识 |
| **战斗属性** | 攻击力、防御力、速度、暴击率、闪避率、穿透 |
| **生活属性** | 魅力、领导力、炼丹天赋、炼器天赋、种植天赋、医术 |
| **隐藏属性** | 气运、命格、隐藏天赋、心魔、道心 |

### 4.2 属性影响直观化

| 方式 | 说明 |
|------|------|
| **数值预览** | 根骨 85（修炼速度+42%） |
| **图标提示** | ⚔️影响战斗、📖影响学习 |
| **详情面板** | 点击展开详细影响 |

### 4.3 属性成长

| 成长方式 | 说明 |
|---------|------|
| **修炼成长** | 日常修炼、闭关、突破、功法修炼 |
| **资源成长** | 丹药、天材地宝、灵液 |
| **经历成长** | 战斗、任务、探索、社交、危机 |

### 4.4 成长限制

| 限制类型 | 说明 |
|---------|------|
| **硬上限** | 资质属性上限100-150 |
| **软上限** | 达到阈值后成长效率降低 |
| **条件突破** | 境界/功法/资源可突破上限 |
| **成长方式限制** | 单次/每日/每月/终身上限 |
| **抗性递减** | 连续使用效果递减 |

### 4.5 AI行为系统

| 组成部分 | 说明 |
|---------|------|
| **个性系统** | 野心、勤勉、忠诚、善良、勇气、谨慎 |
| **决策机制** | 多因素决策、分层决策、记忆系统、学习机制 |
| **行为种类** | 修炼、生产、工作、社交、探索、战斗、休息、消费 |
| **目标系统** | 短期/中期/长期/终极目标 |

### 4.6 角色层级

| 层级 | 职责 | 权限 |
|------|------|------|
| **掌门** | 战略决策、外交、飞升规划 | 全权 |
| **长老** | 部门管理、弟子培养、资源调配 | 部门内 |
| **亲传弟子** | 高级任务、指导内门、参与决策 | 中高级 |
| **内门弟子** | 修炼、任务执行、设施使用 | 基础 |

### 4.7 权限可见性

| 方式 | 说明 |
|------|------|
| **视觉标识** | 掌门金色👑、长老紫色⭐、亲传蓝色💎、内门白色○ |
| **行为差异** | 掌门主动决策、长老部门管理、亲传重要任务、内门日常执行 |
| **交互反馈** | 不同层级不同反馈风格 |
| **事件触发** | 不同层级可触发不同事件 |

### 4.8 角色关系

| 关系类型 | 说明 |
|---------|------|
| **师徒关系** | 指导和学习关系 |

## 六、战斗系统

### 6.1 战斗定位

| 方面 | 说明 |
|------|------|
| **战斗定位** | 实力的表现，非游戏重心 |
| **战斗目的** | 验证角色实力、解决冲突、获取资源 |
| **控制方式** | AI自动控制，玩家观看 |
| **战斗结果** | 真实计算，非模拟 |

### 6.2 战斗机制

| 方面 | 说明 |
|------|------|
| **战斗模式** | 标准回合制战斗 |
| **队伍人数** | 1-5人组队 |
| **实力计算** | 基础实力50%+属性实力30%+功法实力15%+装备实力5% |

### 6.3 属性作用

| 属性 | 作用 |
|------|------|
| **攻击力** | 直接影响每次攻击的伤害数值 |
| **防御力** | 直接减少受到的伤害数值 |
| **速度** | 决定每回合的行动顺序 |
| **暴击率** | 每次攻击独立判定是否暴击 |
| **闪避率** | 每次被攻击独立判定是否闪避 |
| **穿透** | 计算伤害时无视部分防御 |

### 6.4 战斗类型

| 类型 | 内容 |
|------|------|
| **PVE战斗** | 试炼战斗、探索战斗、守护战斗、历练战斗、任务战斗 |
| **PVP战斗** | 切磋、竞技、挑战、宗门战、复仇战 |
| **特殊战斗** | 限时战斗、生存战斗、Boss战、守护战、逃脱战 |

### 6.5 战斗奖励

| 奖励类型 | 说明 |
|---------|------|
| **经验奖励** | 提升角色经验 |
| **资源奖励** | 灵石、材料等 |
| **物品奖励** | 装备、丹药等 |
| **声望奖励** | 提升声望值 |
| **贡献奖励** | 宗门贡献点 |
| **特殊奖励** | 功法、秘籍等 |

### 6.6 战斗评价

| 评价 | 条件 | 奖励加成 |
|------|------|---------|
| 完美 | 无伤亡、回合<5 | +100% |
| 优秀 | 无伤亡、回合<10 | +50% |
| 良好 | 伤亡<20%、回合<15 | +20% |
| 普通 | 正常胜利 | +0% |
| 惨胜 | 伤亡>50% | -30% |

### 6.7 战斗AI

| 方面 | 说明 |
|------|------|
| **决策策略** | 进攻、防守、集火、控制、支援 |
| **AI个性** | 勇敢、谨慎、聪明、鲁莽、保守、团队 |
| **适应性** | 根据战况自动调整策略 |

## 七、关键设计理念

### 7.1 时间单位

| 单位 | 换算 |
|------|------|
| **游戏最小单位** | 季度 |
| **游戏1年** | 现实6分钟 |

### 7.2 AI角色层级

| 层级 | 说明 |
|------|------|
| **掌门** | AI角色 |
| **长老** | AI角色 |
| **亲传弟子** | AI角色 |
| **内门弟子** | AI角色 |
| **外门弟子及以下** | 可重用资源 |

### 7.3 核心设计原则

1. **玩家作为经济参与者** - 玩家在市场中买卖、调控税率、控制历练
2. **战斗是实力表现** - 非游戏重心，AI自动控制，结果真实
3. **贡献点是补充经济** - 灵石是核心，贡献点严格控制
4. **权限需要可见** - 通过视觉、行为、交互、事件多维度体现
5. **成长有限制** - 硬上限+软上限+条件突破+方式限制

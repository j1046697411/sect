# 代码地图 - AI 指引

## 目录定位
存放代码模块与 API 的实时映射地图。

## 核心职责
- 建立代码文件与业务逻辑的数字化关联。
- 维护导出 API 及模块依赖关系的文档。

## AI 开发指引
- **同步性**: 按照 `AGENTS.md` 中的“代码地图规范”，变更代码必须更新地图。
- **准确性**: 确保文件路径和导出 API 的描述与实际代码完全匹配。

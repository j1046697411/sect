# 弟子管理页面布局设计

## 一、设计原则

1. **信息层次分明**：优先展示弟子核心数据和状态
2. **筛选功能强大**：提供多种筛选条件，便于管理大量弟子
3. **操作便捷高效**：提供批量操作和快捷入口
4. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
5. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
6. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 二、页面布局结构

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理（当前选中）、资源管理、设施建设
- 功能区域：地图探索、任务大厅、功法堂
- 生产系统：炼丹房、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示弟子管理详细内容
**详细内容**：

#### 3.1 弟子统计
- 弟子总数、各职务弟子数量
- 各境界弟子数量
- 直观的数据统计，便于了解宗门弟子整体情况

#### 3.2 筛选与管理
- 多种筛选条件：按职务（外门、内门、亲传、杂役）、按境界（炼气、筑基、金丹）
- 多种排序方式：按资质、修为、年龄、贡献、忠诚度、勤勉度
- 批量操作功能：招募弟子、批量分配任务、批量培养、刷新列表、导出数据

#### 3.3 弟子列表
- 表格形式展示弟子信息，包含：姓名、职务、境界、年龄、资质、忠诚度、勤勉度、贡献、状态
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的弟子
- 状态实时更新：修炼中、巡逻中、炼丹中、闭关、任务中、休息中

#### 3.4 分页与统计
- 分页显示，每页10人
- 支持上一页、下一页、首页、末页导航
- 支持直接跳转至指定页码
- 显示当前页码、总页数、每页人数、总人数

#### 3.5 功能标签页
- 弟子列表（当前选中）、详细信息、培养计划、任务分配、成长轨迹
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示当前选中弟子信息和快捷操作
**详细内容**：

#### 4.1 弟子基本信息
- 弟子名称、境界
- 气血、灵力、寿元、贡献
- 状态加成

#### 4.2 弟子快捷操作
- 查看详情、分配任务、培养计划、调整职务
- 赐下丹药、传授功法、驱逐宗门、刷新信息
- 常用操作的快捷入口

#### 4.3 弟子资质
- 根骨、悟性、福缘、心性
- 野心、善良、勇气、谨慎
- 全面展示弟子的资质属性

#### 4.4 当前任务
- 弟子当前执行的任务名称和进度
- 任务状态实时更新

#### 4.5 成长轨迹
- 弟子的修为成长历程
- 直观展示弟子的成长路径

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 四、交互设计

### 1. 快捷键系统

- 顶部导航栏支持展开详情
- 底部状态栏支持折叠/展开

### 2. 动态布局调整

- 对话框模态显示

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

# 任务大厅页面布局设计

## 一、设计原则

1. **任务分类清晰**：将任务按类型分组展示，便于管理
2. **核心信息突出**：优先展示任务的类型、要求、奖励和状态
3. **任务流程直观**：清晰展示任务的接受、执行、完成流程
4. **操作便捷高效**：提供任务接受、放弃、查看详情、加速等快捷操作
5. **任务统计全面**：显示任务的接受情况、完成情况和最新任务
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索、任务大厅（当前选中）、功法堂
- 生产系统：炼丹房、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示任务大厅详细内容
**详细内容**：

#### 3.1 任务筛选
- 多种分类方式：主线、日常、委托、试炼、宗门、个人
- 多种难度筛选：无要求、炼气、筑基、金丹、元婴、化神
- 多种状态筛选：可接受、进行中、已完成、已放弃、已过期
- 便于玩家快速找到符合条件的任务

#### 3.2 任务列表
- 按类型展示任务，包含：任务类型、任务名称、难度、要求、奖励、状态
- 不同任务类型使用不同图标和颜色区分
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的任务
- 状态实时更新：可接受、进行中、已完成、已放弃、已过期
- 提供快捷操作按钮：接受、放弃、查看详情、加速、领取奖励

#### 3.3 功能标签页
- 任务列表（当前选中）、任务详情、任务日志、任务统计、任务建议
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示任务统计和快捷操作
**详细内容**：

#### 4.1 任务统计
- 已接任务数量和上限
- 可接任务数量和上限
- 完成任务数量和总任务数量

#### 4.2 任务快捷操作
- 常用操作：任务列表、任务统计、刷新、任务日志、任务建议、快速接受、快速完成
- 快捷入口，提升操作效率

#### 4.3 今日完成
- 显示今日已完成的任务数量和上限
- 按任务类型分类统计

#### 4.4 最新任务
- 显示最新发布的任务
- 滚动显示，支持快速查看

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 五、响应式设计

### 1. 可折叠区域

- 右侧信息区支持折叠/展开
- 左侧菜单栏支持折叠/展开
- 顶部导航栏支持展开详情
- 底部状态栏支持折叠/展开

### 2. 动态布局调整

- 根据内容自动调整高度
- 长列表支持滚动
- 对话框模态显示
- 自适应不同屏幕尺寸

- 任务大厅页面，管理宗门和个人任务
- 掌门模式，统筹管理宗门任务分配
- 长老模式，管理特定领域的任务
- 弟子模式，接受和完成个人任务

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

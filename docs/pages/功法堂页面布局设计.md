# 功法堂页面布局设计

## 一、设计原则

1. **功法分类清晰**：将功法按属性、类型、等级分组展示，便于学习和管理
2. **核心信息突出**：优先展示功法的名称、属性、等级、要求和效果
3. **学习流程直观**：清晰展示功法的学习条件、消耗和时间
4. **操作便捷高效**：提供功法学习、修炼、传承、放弃等快捷操作
5. **功法统计全面**：显示已学习功法数量、可学习功法数量和最新功法
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索、任务大厅、功法堂（当前选中）
- 生产系统：炼丹房、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示功法堂详细内容
**详细内容**：

#### 3.1 功法筛选
- 多种属性筛选：全部、金、木、水、火、土、风、雷
- 多种类型筛选：修炼、攻伐、防御、辅助、特殊
- 多种等级筛选：凡级、灵级、地级、天级、神级
- 多种状态筛选：可学习、已学习、修炼中、已掌握
- 便于玩家快速找到符合条件的功法

#### 3.2 功法列表
- 按属性和等级展示功法，包含：功法名称、属性、等级、类型、要求、消耗、效果、状态
- 不同等级功法使用不同图标和颜色区分
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的功法
- 状态实时更新：可学习、已学习、修炼中、已掌握
- 提供快捷操作按钮：学习、查看详情、放弃、修炼、传授、继续、加速

#### 3.3 功能标签页
- 功法列表（当前选中）、功法详情、学习记录、修炼进度、传承管理
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示功法统计和快捷操作
**详细内容**：

#### 4.1 功法统计
- 功法总数
- 已学习功法数量
- 可学习功法数量

#### 4.2 功法快捷操作
- 常用操作：功法列表、功法统计、刷新、学习记录、学习建议、快速学习、快速修炼
- 快捷入口，提升操作效率

#### 4.3 已学习功法
- 显示当前已学习的功法名称和等级
- 快速查看已学习的功法

#### 4.4 修炼中功法
- 显示正在修炼的功法名称和进度
- 实时更新修炼进度

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 四、交互设计

- 选中项高亮显示

- 操作成功/失败提示
- 功法状态变化实时更新
- 修炼进度实时更新
- 新功法解锁提示

## 五、响应式设计

- 右侧信息区支持折叠/展开

- 顶部导航栏支持展开详情
- 底部状态栏支持折叠/展开

### 2. 动态布局调整

- 根据内容自动调整高度
- 长列表支持滚动
- 对话框模态显示
- 自适应不同屏幕尺寸

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

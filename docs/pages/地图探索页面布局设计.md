# 地图探索页面布局设计

## 一、设计原则

1. **区域可视化**：直观展示游戏世界的区域分布和状态
2. **核心数据突出**：优先展示区域的资源产出、难度和占领状态
3. **探索流程清晰**：清晰展示探索和占领的条件、资源需求和时间
4. **操作便捷高效**：提供探索、占领、管理等快捷操作
5. **事件和资源可视化**：显示区域事件和资源分布
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索（当前选中）、任务大厅、功法堂
- 生产系统：炼丹房、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示地图探索详细内容
**详细内容**：

#### 3.1 区域统计
- 总区域数、已探索区域数、已占领区域数、可探索区域数、未探索区域数
- 大区域和小区域的数量
- 直观的数据统计，便于了解游戏世界的探索情况

#### 3.2 区域分类与筛选
- 多种分类方式：全部、大区域、小区域、灵脉、古战场、秘境、妖兽森林、凡人国度
- 多种筛选条件：难度、资源、状态、类型、距离
- 便捷操作：探索区域、占领区域、管理队伍、刷新列表、导出数据

#### 3.3 区域列表
- 表格形式展示区域信息，包含：区域名称、类型、等级、状态、资源、难度、距离、占领度、探索度
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的区域
- 状态实时更新：已占领、已探索、可探索、未探索

#### 3.4 区域详情
- 区域名称、类型、等级、状态
- 资源产出、难度、距离、占领度、探索度
- 特殊事件和资源
- 详细的区域信息，便于玩家决策

#### 3.5 功能标签页
- 区域列表（当前选中）、区域详情、探索队伍、占领管理、事件记录
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示探索统计和快捷操作
**详细内容**：

#### 4.1 探索统计
- 已探索区域数量
- 已占领区域数量
- 正在探索的区域数量

#### 4.2 探索快捷操作
- 常用操作：探索、占领、管理、刷新、任务、试炼、资源、事件
- 快捷入口，提升操作效率

#### 4.3 探索建议
- 根据宗门当前状态，提供智能探索建议
- 优先级标记，引导用户决策

#### 4.4 探索队伍
- 探索队伍数量
- 空闲队伍数量

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

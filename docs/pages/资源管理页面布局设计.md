# 资源管理页面布局设计

## 一、设计原则

1. **资源分类清晰**：将资源按类型分组展示，便于管理
2. **核心数据突出**：优先展示资源当前数量和上限
3. **操作便捷高效**：提供资源分配、交易、转换等快捷操作
4. **资源流动可视化**：显示资源的收入和支出情况
5. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
6. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
7. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 二、页面布局结构

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理（当前选中）、设施建设
- 功能区域：地图探索、任务大厅、功法堂
- 生产系统：炼丹房、炼器房
- 系统：社交、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示资源管理详细内容
**详细内容**：

#### 3.1 资源统计
- 核心资源：灵石、贡献点、声望
- 基础资源：灵草、矿石、木材、泉水
- 显示当前数量和上限
- 直观的数据统计，便于了解宗门资源整体情况

#### 3.2 资源分类与筛选
- 多种分类方式：全部、基础资源、高级资源、丹药、装备、功法
- 多种筛选条件：数量、类型、品质、用途、获取方式
- 便捷操作：分配资源、资源交易、资源转换、刷新列表、导出数据

#### 3.3 资源列表
- 表格形式展示资源信息，包含：资源名称、类型、当前数量、最大容量、品质、用途、价值、状态
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的资源
- 状态实时更新：充足、有限、稀少

#### 3.4 资源流动日志
- 显示最近的资源流动记录
- 包含时间、资源变化、变化原因
- 支持滚动查看更多日志

#### 3.5 资源分配建议
- 根据宗门当前状态，提供智能资源分配建议
- 优先级标记，引导用户决策

#### 3.6 功能标签页
- 资源概览（当前选中）、仓库管理、资源流动、交易记录、分配记录
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示仓库信息和快捷操作
**详细内容**：

#### 4.1 仓库基本信息
- 仓库容量、物品总数
- 实时更新仓库状态

#### 4.2 仓库快捷操作
- 仓库管理、刷新、取出、存入
- 交易、转换、分配、消耗
- 常用操作的快捷入口

#### 4.3 资源流动
- 今日收入、今日支出
- 直观展示资源的每日流动情况

#### 4.4 最新入库
- 最近入库的物品列表
- 显示物品名称、数量、入库时间
- 滚动显示最新入库记录

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局

# 社交页面布局设计

## 一、设计原则

1. **关系分类清晰**：将社交关系按类型、友好度分组展示，便于管理和互动
2. **核心信息突出**：优先展示关系的名称、类型、友好度和状态
3. **互动流程直观**：清晰展示互动的条件、消耗和效果
4. **操作便捷高效**：提供互动、交流、宣战、联盟等快捷操作
5. **关系统计全面**：显示友好关系、敌对关系、中立关系的数量
6. **视觉层次清晰**：通过边框、分隔线和缩进区分不同功能区域
7. **响应式设计**：支持动态布局调整，适应不同屏幕尺寸
8. **快捷键支持**：提供常用操作的快捷键，提升操作效率

## 三、区域功能详细说明

### 1. 顶部导航栏

**位置**：界面顶部，3行×160字符
**功能**：显示核心游戏信息，提供快速访问和全局搜索
**详细内容**：
- 第一行：宗门名称、地点、天气、游戏时间、游戏速度、全局搜索框
- 第二行：宗门关键数据（弟子总数、资源、设施、占领区域等）
- 第三行：功能分类导航（核心管理、功能区域、生产系统、系统）

### 2. 左侧菜单栏

**位置**：界面左侧，52行×30字符
**功能**：提供游戏主要功能的快速导航
**详细内容**：
- 核心管理：宗门总览、弟子管理、资源管理、设施建设
- 功能区域：地图探索、任务大厅、功法堂
- 生产系统：炼丹房、炼器房
- 系统：社交（当前选中）、设置
- 使用数字快捷键1-0,S快速访问

### 3. 中央内容区

**位置**：界面中央，34行×100字符
**功能**：展示社交详细内容
**详细内容**：

#### 3.1 关系统计
- 友好宗门数量
- 敌对宗门数量
- 中立宗门数量
- 弟子关系数量
- 访客数量
- 直观的数据统计，便于了解宗门社交关系整体情况

#### 3.2 关系分类与筛选
- 多种分类方式：全部、宗门、弟子、访客、特殊
- 多种状态筛选：友好、敌对、中立、陌生人、盟友、敌对
- 多种友好度筛选：0-20、21-40、41-60、61-80、81-100
- 便于玩家快速找到符合条件的社交关系

#### 3.3 关系列表
- 按友好度展示关系，包含：关系名称、类型、友好度、关系状态、贡献、快捷操作
- 不同关系类型使用不同图标和颜色区分
- 支持上下箭头导航，Enter键查看详情
- 高亮显示当前选中的关系
- 提供快捷操作按钮：交流、结盟、礼物、宣战、和解

#### 3.4 功能标签页
- 关系列表（当前选中）、关系详情、互动记录、礼物管理、外交管理
- 支持Tab键切换

### 4. 右侧信息区

**位置**：界面右侧，15行×30字符
**功能**：显示社交统计和快捷操作
**详细内容**：

#### 4.1 社交统计
- 友好宗门数量
- 敌对宗门数量
- 中立宗门数量

#### 4.2 社交快捷操作
- 常用操作：交流、结盟、宣战、发送礼物、刷新、关系记录、关系建议
- 快捷入口，提升操作效率

#### 4.3 今日互动
- 交流次数和上限
- 礼物次数和上限
- 宣战次数和上限

#### 4.4 最新消息
- 显示最新的社交消息
- 滚动显示，支持快速查看

### 5. 底部状态栏

**位置**：界面底部，5行×160字符
**功能**：显示系统信息和快捷键提示
**详细内容**：
- 第一行：新任务、弟子传讯、宗门事件、自动存档状态、帮助、菜单、当前模式、游戏速度、运行时间、版本号
- 第二行：快捷键提示
- 第三行：消息提示
- 第四行：系统状态（内存使用、CPU使用率、网络、存档、日志、音效、音乐）
- 第五行：开发者信息

## 四、交互设计

| Ctrl+C | 交流 | 关系列表 |
| Ctrl+A | 结盟 | 关系列表 |
| Ctrl+D | 宣战 | 关系列表 |
| Ctrl+G | 发送礼物 | 关系列表 |

### 2. 交互反馈

- 选中项高亮显示
- 按钮悬停效果
- 操作成功/失败提示
- 关系状态变化实时更新
- 新消息提示
- 友好度变化提示

## 五、响应式设计

### 1. 可折叠区域

- 右侧信息区支持折叠/展开
- 左侧菜单栏支持折叠/展开
- 顶部导航栏支持展开详情
- 底部状态栏支持折叠/展开

### 2. 动态布局调整

- 根据内容自动调整高度
- 长列表支持滚动
- 对话框模态显示
- 自适应不同屏幕尺寸

## 七、适用场景

- 社交页面，管理宗门和个人社交关系
- 掌门模式，统筹管理宗门外交关系
- 长老模式，管理特定领域的社交事务
- 弟子模式，管理个人社交关系

## 八、实现建议

1. **模块化开发**：将中央内容区的各个模块作为独立组件开发
2. **状态管理**：使用状态管理库管理界面状态
3. **响应式布局**：实现动态布局调整
4. **快捷键支持**：实现完整的快捷键系统
5. **性能优化**：优化长列表滚动性能
6. **测试**：进行多场景测试，确保布局在不同情况下正常显示
7. **用户反馈**：收集用户反馈，持续优化布局
